Getting Started with Flash Sheet
===================================

Welcome to Flash Sheet, a powerful data analysis and export tool with advanced features for data manipulation and visualization.

What is Flash Sheet?
--------------------

Flash Sheet is a comprehensive application designed to help you:

- Load and analyze data from multiple sources (CSV, Excel, JSON, SQL databases)
- Visualize data through interactive tables and charts
- Export data in various formats (PDF, Excel, CSV, SQL, Images)
- Perform advanced operations like data joins and pivot tables
- Create separated exports using Excel templates

Key Features
------------

- **Multi-format Data Loading**: Support for CSV, Excel (.xlsx, .xls), JSON, Parquet, HDF5, and SQL databases
- **Interactive Data Visualization**: Tables with pagination, filtering, and sorting
- **Chart Generation**: Multiple chart types for data analysis
- **Advanced Export Options**: PDF, Excel, CSV, SQL, and image formats
- **Data Manipulation**: Joins, pivot tables, and folder loading capabilities
- **Template-based Exports**: Preserve Excel formatting with custom templates

System Requirements
-------------------

- **Operating System**: Windows 10+, macOS 10.15+, Linux (Ubuntu 18.04+)
- **Python**: 3.7 or higher
- **Memory**: Minimum 4GB RAM (8GB recommended for large datasets)
- **Storage**: 500MB free space for installation

Installation
------------

1. Download the latest release from the releases page
2. Run the installer appropriate for your operating system
3. Follow the installation wizard
4. Launch Flash Sheet from your applications menu

For detailed installation instructions, see :doc:`../releases/installation_guide`.

First Steps
-----------

1. **Launch the Application**
   - Open Flash Sheet from your desktop or applications menu
   - The main interface will appear with the data loading options

2. **Load Your First Dataset**
   - Click on "Cargar Archivo" in the main menu
   - Select a CSV or Excel file from your computer
   - The data will be loaded and displayed in the main table view

3. **Explore the Interface**
   - Use the navigation tabs to switch between different views
   - Try the filtering and sorting options in the data table
   - Experiment with basic export functions
