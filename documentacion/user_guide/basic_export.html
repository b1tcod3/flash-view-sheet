Guía de Exportación Básica

Flash Sheet proporciona capacidades comprehensivas de exportación para guardar tus datos y visualizaciones en varios formatos para uso en otras aplicaciones.

Formatos de Exportación Soportados

Formatos de Exportación de Datos

  • CSV (Valores Separados por Coma): Formato universal de hoja de cálculo

  • Excel (.xlsx): Formato Microsoft Excel con formato

  • JSON: Formato de datos estructurado para aplicaciones web

  • Parquet: Formato columnar para big data

  • HDF5: Formato de datos científicos

  • SQL: Declaraciones de inserción de base de datos

Formatos de Exportación de Visualización

  • PDF: Formato de documento portable para reportes

  • PNG: Formato de imagen de alta calidad

  • SVG: Gráficos vectoriales escalables

  • JPEG: Formato de imagen comprimida

Exportación de Tablas de Datos

Exportación CSV

  1. Haz clic en “Exportar” → “Exportar CSV”

  2. Elige la carpeta de destino y nombre de archivo

  3. Configura opciones: - Delimitador: Coma, punto y coma, tabulación, o personalizado - Codificación: UTF-8, ISO-8859-1, u otra - Incluir Encabezados: Incluir nombres de columna como primera fila - Carácter de Comillas: Carácter para encerrar campos

  4. Haz clic en “Exportar”

Exportación Excel

  1. Haz clic en “Exportar” → “Exportar Excel”

  2. Selecciona destino y nombre de archivo

  3. Elige opciones: - Nombre de Hoja: Nombre para la hoja de trabajo Excel - Incluir Formato: Preservar formato de tabla - Auto-ajustar Columnas: Ajustar anchos de columna automáticamente - Congelar Encabezados: Mantener encabezados de columna visibles al desplazarse

  4. Haz clic en “Exportar”

Exportación JSON

  1. Haz clic en “Exportar” → “Exportar JSON”

  2. Configura opciones de formato: - Imprimir Bonito: JSON formateado con indentación - Compacto: JSON minificado para archivos más pequeños - Formato Array: Exportar como array de objetos JSON - Formato Object: Exportar como objeto JSON único

Exportación SQL

  1. Haz clic en “Exportar” → “Exportar SQL”

  2. Selecciona tipo de base de datos (MySQL, PostgreSQL, SQLite, etc.)

  3. Configura opciones: - Nombre de Tabla: Nombre para la tabla de base de datos - Crear Tabla: Incluir declaración CREATE TABLE - Eliminar Tabla: Incluir DROP TABLE IF EXISTS - Método de Inserción: INSERT único o bulk INSERT

  4. Haz clic en “Exportar”

Exportación de Gráficos y Visualizaciones

Exportación PDF

  1. Cambia a Vista de Gráficos

  2. Selecciona el gráfico a exportar

  3. Haz clic en “Exportar” → “Exportar PDF”

  4. Configura: - Tamaño de Página: A4, Carta, Legal, Personalizado - Orientación: Retrato o Paisaje - Márgenes: Márgenes de página en pulgadas/cm - Calidad: Configuraciones de resolución

Exportación de Imagen

  1. Selecciona el gráfico en Vista de Gráficos

  2. Haz clic en “Exportar” → “Exportar Imagen”

  3. Elige formato (PNG, JPEG, SVG)

  4. Configura: - Resolución: DPI (72, 150, 300, personalizado) - Tamaño: Ancho y alto en píxeles - Fondo: Transparente o color sólido - Calidad: Nivel de compresión (para JPEG)

Opciones de Exportación y Configuraciones

Selección de Rango de Datos

  • Todos los Datos: Exportar el conjunto de datos completo

  • Vista Actual: Exportar solo datos visibles/filtrados

  • Filas Seleccionadas: Exportar solo filas seleccionadas

  • Rango Personalizado: Especificar rango de filas a exportar

Selección de Columnas

  • Todas las Columnas: Exportar todas las columnas

  • Columnas Visibles: Solo columnas actualmente visibles

  • Columnas Seleccionadas: Elegir columnas específicas

  • Orden Personalizado: Reordenar orden de columnas para exportación

Nomenclatura de Archivos y Organización

Nomenclatura Automática

  • Timestamp: Incluir fecha/hora en nombre de archivo

  • Nombre de Dataset: Usar nombre de dataset original

  • Tipo de Exportación: Incluir formato en nombre de archivo

  • Números Secuenciales: Para múltiples exportaciones

Plantillas Personalizadas

Usa placeholders para nomenclatura dinámica:

  • {date}: Fecha actual (YYYY-MM-DD)

  • {time}: Hora actual (HHMMSS)

  • {dataset}: Nombre del dataset

  • {type}: Formato de exportación

  • {counter}: Número secuencial

Exportación por Lotes

  • Múltiples Formatos: Exportar a varios formatos simultáneamente

  • Archivos Divididos: Dividir datasets grandes en múltiples archivos

  • Aplicación de Plantilla: Aplicar diferentes configuraciones por formato

Consideraciones de Rendimiento

Manejo de Datasets Grandes

  • Chunking: Dividir automáticamente exportaciones grandes

  • Gestión de Memoria: Procesamiento eficiente para big data

  • Seguimiento de Progreso: Monitorear progreso de exportación

  • Cancelación: Detener exportaciones de larga duración

Consejos de Optimización

  • Filtrar Primero: Aplicar filtros antes de exportar para reducir tamaño

  • Seleccionar Columnas: Exportar solo columnas necesarias

  • Elegir Formato: Seleccionar formato apropiado para tu caso de uso

  • Compresión: Usar formatos comprimidos cuando sea posible

Solución de Problemas de Exportación

Problemas Comunes

  • Archivo Bloqueado: Cerrar archivo si está abierto en otra aplicación

  • Permiso Denegado: Verificar permisos de escritura en carpeta destino

  • Espacio en Disco: Asegurar espacio suficiente para exportación

  • Problemas de Codificación: Elegir codificación de texto apropiada

Recuperación de Errores

  • Exportaciones Parciales: Reanudar exportaciones interrumpidas

  • Creación de Backup: Backup automático de archivos existentes

  • Validación: Verificar integridad de exportación

  • Archivos de Log: Revisar logs detallados de errores
