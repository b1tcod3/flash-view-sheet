Guía de Gráficos y Visualizaciones

Flash Sheet proporciona capacidades comprehensivas de gráficos para visualizar tus datos a través de varios tipos de gráficos y opciones de personalización.

Tipos de Gráficos Disponibles

Gráficos de Barras

  • Barras Verticales: Gráfico de barras estándar para comparar categorías

  • Barras Horizontales: Barras mostradas horizontalmente

  • Barras Apiladas: Múltiples series de datos apiladas

  • Barras Agrupadas: Comparación lado a lado de múltiples series

Gráficos de Líneas

  • Línea Simple: Conectar puntos de datos con líneas

  • Múltiples Líneas: Comparar múltiples series de datos

  • Gráficos de Área: Área rellena bajo la línea

  • Gráficos Spline: Líneas curvas suaves

Gráficos de Torta

  • Torta Estándar: Gráfico circular dividido por proporciones

  • Gráficos de Dona: Gráfico de torta con centro removido

  • Torta Explotada: Segmentos jalados para énfasis

Gráficos de Dispersión

  • Dispersión XY: Trazar puntos basados en dos variables numéricas

  • Gráficos de Burbujas: Agregar tercera dimensión con tamaño de burbuja

  • Líneas de Tendencia: Agregar líneas de regresión y curvas

Otros Tipos de Gráficos

  • Histogramas: Distribución de datos numéricos

  • Gráficos de Caja: Resumen de distribución estadística

  • Mapas de Calor: Visualización de matriz con codificación de color

Creando Gráficos

Creación Básica de Gráficos

  1. Cambia a la pestaña Vista de Gráficos

  2. Selecciona tu tipo de gráfico de la barra de herramientas

  3. Elige las columnas de datos para ejes X e Y

  4. Personaliza apariencia y opciones

  5. Exporta o guarda el gráfico

Selección de Datos

  • Selección de Columnas: Elegir qué columnas usar para el gráfico

  • Rango de Datos: Seleccionar filas específicas o usar datos filtrados

  • Agregación: Agrupar datos por categorías o períodos de tiempo

  • Muestreo: Usar muestreo de datos para datasets muy grandes

Personalización de Gráficos

Opciones de Apariencia

  • Colores: Elegir de paletas predefinidas o colores personalizados

  • Temas: Aplicar diferentes temas visuales

  • Fuentes: Personalizar fuentes de título, eje y etiqueta

  • Líneas de Cuadrícula: Mostrar/ocultar cuadrícula de fondo

Configuración de Ejes

  • Etiquetas de Ejes: Personalizar títulos de ejes

  • Opciones de Escala: Lineal, logarítmica o escalas personalizadas

  • Marcas de Graduación: Controlar frecuencia de marcas de graduación y formato

  • Límites de Rango: Establecer valores mínimo y máximo

Leyenda y Títulos

  • Título del Gráfico: Agregar títulos descriptivos

  • Títulos de Ejes: Etiquetar ejes X e Y apropiadamente

  • Leyenda: Posicionar y personalizar la leyenda

  • Etiquetas de Datos: Mostrar valores en elementos del gráfico

Características Avanzadas

Gráficos Interactivos

  • Tooltips: Información al pasar el mouse sobre puntos de datos

  • Zooming: Hacer zoom en áreas del gráfico

  • Panning: Navegar gráficos grandes

  • Selección: Hacer clic para seleccionar puntos de datos

Superposiciones Estadísticas

  • Líneas de Tendencia: Lineal, polinomial o tendencias exponenciales

  • Barras de Error: Mostrar variabilidad de datos

  • Intervalos de Confianza: Bandas de confianza estadística

  • Promedios Móviles: Suavizar tendencias de datos

Gráficos Múltiples

  • Subplots: Múltiples gráficos en una vista

  • Grids de Facetas: Gráficos divididos por categorías

  • Gráficos Comparativos: Comparaciones lado a lado

Exportando Gráficos

Formatos Soportados

  • PNG: Imagen raster de alta calidad

  • SVG: Gráficos vectoriales escalables

  • PDF: Formato vectorial para documentos

  • JPEG: Formato de imagen comprimida

Opciones de Exportación

  • Resolución: Configuraciones DPI para calidad de imagen

  • Tamaño: Dimensiones personalizadas

  • Fondo: Fondo transparente o color sólido

  • Compresión: Compensaciones calidad vs tamaño de archivo

Plantillas de Gráficos

  • Guardar Plantillas: Reutilizar configuraciones de gráficos

  • Cargar Plantillas: Aplicar configuraciones guardadas

  • Configuraciones Predeterminadas: Establecer preferencias personales

Consideraciones de Rendimiento

Para datasets grandes:

  • Muestreo de Datos: Muestrear automáticamente datasets grandes

  • Renderizado Progresivo: Renderizar gráficos incrementalmente

  • Optimización de Memoria: Uso eficiente de memoria

  • Almacenamiento en Caché: Almacenar en caché gráficos renderizados

Mejores Prácticas

Principios de Diseño de Gráficos

  • Propósito Claro: Cada gráfico debería tener un mensaje específico

  • Tipo Apropiado: Elegir el gráfico correcto para tus datos

  • Simplicidad: Evitar desorden y elementos innecesarios

  • Accesibilidad: Considerar daltonismo y legibilidad

Preparación de Datos

  • Limpiar Datos: Remover valores atípicos o manejar valores faltantes

  • Escalas Apropiadas: Elegir escalas que muestren datos claramente

  • Formatos Consistentes: Usar formatos consistentes de fecha/número

  • Etiquetas Significativas: Usar etiquetas descriptivas de eje y leyenda
